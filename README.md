# Summary 
This program will parse all files within a directory to generate a hashed version of the file.

Files to be parsed are here:  "\CSharpTraining\Hawthorn Xerox Sample OPO files\Input"

### Hash Calculation
For each line the hash value is the sum the ASCII Value of the character multiplied by the position of the character in the line.

### Output
Hashed vesrsions of files will be saved to : "\CSharpTraining\Hawthorn Xerox Sample OPO files\Output"

The filename is suffixed with ‘H’. Example: 

  Source File name : gscopohawthorn.1495038

  Hashed File name : gscopohawthorn.1495038**H**
